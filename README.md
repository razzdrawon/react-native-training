# React & React Native Training

Curso completo de React y React Native avanzado con ejercicios prácticos y ejemplos modernos.

## Estructura del Curso

Este curso está diseñado para desarrolladores que quieren dominar React y React Native con las últimas prácticas y tecnologías.

### Módulos

- **Módulo 0**: Recapitulación Conceptual
- **Módulo 1**: React Hooks Avanzados y Gestión de Estado
- **Módulo 2**: Patrones Avanzados de React
- **Módulo 3**: Routing y Navegación
- **Módulo 4**: Performance y Optimización
- **Módulo 5**: Forms y Validación Avanzada
- **Módulo 6**: React Native Fundamentals
- **Módulo 7**: Testing Avanzado
- **Módulo 8**: Features Avanzadas

## Desarrollo

### Instalación

```bash
npm install
```

### Servidor de Desarrollo

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

Una vez que el servidor esté corriendo, abre tu navegador y navega a `http://localhost:5173/`. La aplicación se recargará automáticamente cuando modifiques cualquier archivo fuente.

### Build

Para compilar el proyecto:

```bash
npm run build
```

Esto compilará tu proyecto y almacenará los artefactos de build en el directorio `dist/`. Por defecto, el build de producción optimiza tu aplicación para performance y velocidad.

### Testing

Para ejecutar los tests:

```bash
npm test
```

Para tests en modo watch:

```bash
npm run test:watch
```
