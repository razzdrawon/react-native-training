# React & React Native Training

Complete advanced React and React Native course with practical exercises and modern examples.

## Course Structure

This course is designed for developers who want to master React and React Native with the latest practices and technologies.

### Modules

- **Module 0**: Conceptual Recap
- **Module 1**: Advanced React Hooks and State Management
- **Module 2**: Advanced React Patterns
- **Module 3**: Routing and Navigation
- **Module 4**: Performance and Optimization
- **Module 5**: Forms and Advanced Validation
- **Module 6**: React Native Fundamentals
- **Module 7**: Advanced Testing
- **Module 8**: Advanced Features

## Development

### Installation

```bash
npm install
```

### Development Server

To start the development server:

```bash
npm run dev
```

Once the server is running, open your browser and navigate to `http://localhost:5173/`. The application will automatically reload whenever you modify any source files.

### Build

To build the project:

```bash
npm run build
```

This will compile your project and store the build artifacts in the `dist/` directory. By default, the production build optimizes your application for performance and speed.

### Testing

To run tests:

```bash
npm test
```

For tests in watch mode:

```bash
npm run test:watch
```
